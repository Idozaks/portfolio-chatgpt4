Project Overview

This Pong game was developed in JavaScript using the p5.js library, focusing on interactive game mechanics and classic arcade gameplay.

Key Features

  • Classic arcade-style Pong gameplay
  • Adjustable game speed and responsive controls
  • Score tracking and dynamic difficulty adjustment
  • Sleek vector graphics for a modern touch
